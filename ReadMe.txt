LocoSoft Brief Read Me

1. Alembic support only available in Maya 2013. This means that Bake Geometry and Load Geometry will
   only work in Maya 2013.
2. AELSSolverNodeTemplate.mel will need to be put into the Maya project's script folder for it to

3. We have precompiled a release version of our plugin for Maya 2013 and it is located in the "precompiled
   plugin" folder.
